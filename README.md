# Dir l'Affaire

Boutique e-commerce Vite + React + TypeScript pour des produits de beauté et de bien-être, avec catalogue, panier, commande et espace admin.

## Installation

```sh
npm install
```

## Développement

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Tests

```sh
npm run test
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
